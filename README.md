# Weather-Map-App
## Description
    Single Page Open weather map app, using OpenWeatherMap weather API and Leaflet map API.
    
    Backend: NodeJs/TypeScript
    
    Frontend: HTML/CSS/JavaScript-ES6
## Install
    git clone https://github.com/BoazRamot/weather-map-app.git
    
    cd weather-map-app
    
    npm install
## Run
    npm start
